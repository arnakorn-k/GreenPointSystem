Green Point System

ปัจจุบันการบริหารจัดการขยะที่เกิดจากการบริโภคในสถานศึกษา เช่น ขวดน้ําพลาสติกและ กระป๋องเครื่องดื่ม กําลังเป็นปัญหาที่สําคัญ เนื่องจากปริมาณขยะที่เพิ่มขึ้นอย่างต่อเนื่องและการขาด การแยกขยะอย่างเหมาะสม ส่งผลให้วัสดุรีไซเคิลเหล่านี้ไม่ถูกนํากลับมาใช้ใหม่และก่อให้เกิดมลพิษต่อ สิ่งแวดล้อมโดยรอบ

Green Point System

ในแผนกเทคโนโลยีสารสนเทศของวิทยาลัยเทคนิคเชียงใหม่ มีจํานวนผู้ใช้บริการสูงในแต่ละวัน ทําให้เกิดขยะจากการบริโภคขวดน้ําและกระป๋องในปริมาณมาก โดยขาดแรงจูงใจหรือระบบที่ สนับสนุนการแยกขยะที่มีประสิทธิภาพ ทําให้ทรัพยากรเหล่านี้สูญเปล่า และส่งผลต่อปัญหา สิ่งแวดล้อมและภาพลักษณ์ของสถานศึกษา

Green Point System

โครงการ "Green Points" จึงถูกพัฒนาขึ้นเพื่อส่งเสริมการแยกขยะประเภทรีไซเคิลในแผนกฯ โดยมีการสร้างระบบสะสมแต้มผ่านเครื่องมือที่ช่วยจดจําและวิเคราะห์ขวดพลาสติกและกระป๋องเพื่อ แลกเป็นรางวัลหรือเงินคืน ซึ่งจะช่วยกระตุ้นพฤติกรรมที่เป็นมิตรต่อสิ่งแวดล้อมและลดปริมาณขยะใน วิทยาลัย

เกี่ยวกับเรา

โครงการ Green Points พัฒนาขึ้นเพื่อส่งเสริมการแยกขยะรีไซเคิลในแผนกเทคโนโลยีสารสนเทศ วิทยาลัยเทคนิคเชียงใหม่ ผ่านระบบสะสมแต้มจากขวดพลาสติกและกระป๋องเครื่องดื่ม เพื่อนำไปแลกรางวัลหรือเงินคืน ระบบนี้ช่วยกระตุ้นพฤติกรรมที่เป็นมิตรต่อสิ่งแวดล้อม ลดปริมาณขยะ และปรับปรุงภาพลักษณ์ของสถานศึกษาให้สะอาดและยั่งยืนยิ่งขึ้น
